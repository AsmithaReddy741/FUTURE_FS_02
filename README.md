# Mini CRM - Client Lead Management System

##  Description
This project is a Mini CRM system that allows users to manage client leads efficiently. It helps track leads, update their status, and store notes for follow-ups.

##  Features
- Add new leads
- View all leads
- Update lead status (New, Contacted, Converted)
- Delete leads
- Add notes for each lead

##  Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

##  How to Run
1. Start backend:
   node server.js

2. Open frontend:
   index.html
